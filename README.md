# Gastro_Epidemio

Software para fins de pesquisa epidemiológica com foco em serviços de colonoscopia.

## Tecnologias

- **Python 3.12+**
- **uv** (gerenciador de pacotes e ambientes)
- **Rich** (interface de terminal)
- **Arch Linux** (ambiente de desenvolvimento principal)

## Como Instalar

Utiliza o gerenciador `uv`. Para preparar o ambiente:

```bash
uv sync
```
